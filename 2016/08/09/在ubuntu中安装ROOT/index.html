在ubuntu中安装ROOT

发表于 2016-08-09   |   分类于 计算机

我本地的ubuntu已经安装好了ROOT，这次想尝试一下在腾讯云主机上安装，熟悉一下安装过程并体验一下远程运行的效果。之前安装过几次ROOT也是情况不断，由于ROOT版本和ubuntu内核版本以及依赖包的问题并不能算完美，虽然最后都能用。在有了一些经验之后往往就不会选择最新的版本，毕竟稳定性和兼容性才是最重要的。

远程图形界面

ROOT需要经常画图，下一个问题就是如何连接远程图形界面。如果直接运行画图命令会出现
DISPLAY not set, setting it to 202.122.36.33:0.0

  • 安装apt-get install x11-utils
  • 使用xmanager的xmanager-passive，连接时在xshell选择转发x11的display为localhost:0.0

就可以看到图形界面连接成功·····xshell看起来也的确不错！此时使用xshell和xftp的ctrl + alt + f新建传输窗口就可以方便地进行文件传输了。
