apt-get被锁

发表于 2016-12-23   |   分类于 计算机

执行命令sudo apt-get update
显示：无法获得锁 /var/lib/apt/lists/lock - open (11: 资源暂时不可用)
搜索得知是因为意外退出apt导致的，于是sudo rm /var/lib/apt/lists/lock进行解锁就可以了。

我的CHIPs

发表于 2016-12-23   |   分类于 计算机

号称9$的CHIP

官网：https://getchip.com/
在网上搜索单板计算机，看到了CHIP这个产品，只要9$。这是基础版的价格，一个HDMI的转换器要15$，所以还是不要了，我并不指望用它来放个视频，上个网什么的，万一需要也可以考虑VNC。看了官网的介绍感觉真心不错，板载蓝牙wifi，内存CPU都还行，体积小，适合随身携带。

谷歌SketchUp

发表于 2016-12-22   |   分类于 计算机


半小时上手

之前从未接触过这个软件，上面的图从安装软件到画好只用了半小时，实际画图也就几分钟；原因只有一个这个软件很方便，适合非专业画图。就像它的名字一样，非常适合初步的结构构思和展现，不要求很好、很精密的设置。

本来打算学SolidWorks，结果下载了半天才下了20%，再想到要安装，破解，然后从头学起；这个还是以后慢慢来吧。想到大三的时候就说要学，结果两三年过去了还是多不多，出来混迟早是要还的。。。。

谷歌出品

在得知这款软件是谷歌出的之后，我对它更有好感了，，，我喜欢谷歌这样的公司，它所有的出品都让人尊敬。
有了这个软件，以后表达一些关于空间设计的东西就方便多了，比如我一直构思一个自己的小实验是，如果能用这个画出来简直完美啊！
